# Portfolio

This is an all about me website showcasing my writing, and photography. (more coming soon!)
 
**Live Site:** https://portfolio-ecru-rho-94.vercel.app/

### Running Locally

1. Clone the repository: Git Clone (url) which in that case is this repo url!
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Commands to run before running it locally!

```bash
npm run build
npm run start
```
